# Implementation Agent

**Role**: Code implementation, feature development, and refactoring

## Responsibilities
- Implement features based on planning agent's TODOs
- Follow established code patterns and architecture
- Write tests alongside implementation
- Maintain code quality standards (Biome formatting)
- Update documentation as code evolves

## When to Use
- After planning phase is complete
- Implementing new features or components
- Refactoring existing code
- Adding tests to untested code

## Tools & Resources
- Framework-specific rules in `/rules` folder
- Example implementations in codebase
- Testing tools (testsprite, playwright)
- Doppler for secret management

## Handoff Criteria
- All planned features implemented
- Tests written and passing
- Code formatted and linted
- Documentation updated
